# Scene V0.1 Manual Smoke Test

喺 real macOS 14+ machine 上面跑。先喺 Xcode 開 `SceneApp/SceneApp.xcodeproj`，⌘R launch。

## Prerequisites

- [ ] macOS 14.0 (Sonoma) 或以上
- [ ] System Settings → Privacy & Security → Accessibility 已清除 Scene 嘅 entry（令 onboarding 可以 fresh 跑）
- [ ] 可選：外接 mon 一個（test P8）
- [ ] 可選：Cursor / VS Code / Slack（test P7）

## Tests

### P1 — 第一次啟動 onboarding

- [ ] 冇 AX 權限狀態下 launch Scene
- [ ] Menu bar 右上見到 Scene icon（`rectangle.3.group`）
- [ ] Click icon → dropdown 只有「Grant Accessibility Access…」+「Quit Scene」
- [ ] Click「Grant Accessibility Access…」→ OnboardingView window 彈出（獨立 NSWindow，有 title bar）
- [ ] Click「Open System Settings」→ 跳 Privacy & Security → Accessibility pane
- [ ] 喺 Settings 加 Scene → toggle on
- [ ] 返 OnboardingView → 最多 2 秒後自動關，menu bar icon 變回正常，dropdown 變 7 個 preset

### P2 — 基本 preset（menu 觸發）

- [ ] 開 3 個 TextEdit 窗口（不同位置都可）
- [ ] Click Scene icon → 揀「Halves」
- [ ] 前 2 個 TextEdit（z-order 最前）分左右半
- [ ] 第 3 個 TextEdit 最小化
- [ ] 確認兩個窗口**冇壓到 menu bar 或 Dock**（`visibleFrame` 正確）

### P3 — Thirds

- [ ] Restore 被最小化嘅窗口
- [ ] 揀「Thirds」→ 3 個窗口分三等寬直欄

### P4 — Hotkey

- [ ] 按 ⌘⇧1 → 頭一個窗口變 full；其餘 minimize
- [ ] 按 ⌘⇧4 → 前 4 個窗口變 2×2 Quads（如果得 3 個窗口，前 3 個佔 slot 1-3，slot 4 留空）

### P5 — 少窗口（underflow）

- [ ] 閂到只剩 2 個 TextEdit
- [ ] 按 ⌘⇧4 → 前 2 個 slot 有窗口，其餘 slot 留空（畫面會見到空位）

### P6 — 零窗口 + notification

- [ ] 閂晒所有 TextEdit（冇任何可見窗口）
- [ ] 按 ⌘⇧2 → **應該**：
  - 如果 Notification permission 已授權：macOS notification banner 彈「Scene — No windows to arrange」
  - 如果未授權：menu bar icon 短暫 dim + tooltip 變 3 秒

### P7 — 權限中途撤銷

- [ ] Scene 運行中，去 System Settings → Accessibility → 關 Scene
- [ ] 返 Scene，最多 2 秒內：
  - Menu bar dropdown 變返「Grant Accessibility Access…」
  - 按 ⌘⇧1-7 冇反應（hotkey unregistered）
- [ ] 重新授權 → 2 秒內 dropdown 變回 7 個 preset

### P8 — Electron tolerance

- [ ] 開 Cursor（或 VS Code / Slack）+ 1-2 個 TextEdit
- [ ] Click Halves → 所有窗口對齊 50/50
- [ ] 用眼量 Cursor 嘅窗口邊界，應該 ±5px 內（唔會見到明顯偏差）

### P9 — Multi-display active screen（如有外接 mon）

- [ ] 滑鼠放 external mon
- [ ] 喺 external mon 開 2 個 TextEdit
- [ ] 按 ⌘⇧2 → **只係** external mon 嘅 TextEdit 左右分；internal mon 嘅窗口**冇郁**

### P10 — visibleFrame 避開 Dock/menu bar

- [ ] Dock 設為 always visible（唔 auto-hide）
- [ ] 按 ⌘⇧1 → 全屏窗口嘅底部**唔會壓到 Dock**、頂部**唔會壓到 menu bar**
